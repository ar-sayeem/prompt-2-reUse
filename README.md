# ♻️ prompt-2-reUse

> A personal collection of reusable AI prompts —
> each one built to eliminate redundant work and save time,
> while producing high-quality, consistent outputs every time.

---

## 💡 Why This Exists

Every time I needed a specific kind of output from AI,
I had to re-explain my style, my format, my preferences — over and over.

That's redundant. So I fixed it.

This repo is my personal prompt library.
Each prompt is carefully crafted, battle-tested, and ready to reuse
whenever I need it — no re-explaining, no reformatting, just results.

## 🚀 How To Use Any Prompt

**Step 1** — Browse the folders and find the prompt you need

**Step 2** — Open its `prompt.md` and copy the full prompt

**Step 3** — Paste it into Claude (claude.ai)

**Step 4** — Follow the input instructions inside the prompt

**Step 5** — Get your output instantly ✅

> Each `prompt.md` contains its own usage instructions inside.
