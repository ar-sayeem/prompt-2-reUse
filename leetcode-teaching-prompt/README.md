# 🧑‍🏫 LeetCode Teaching Prompt

## Why This Was Built

I solved LeetCode problems. But every time I wanted to turn my solution
into a proper teaching post for my beginner students,
I had to re-explain my style, my format, my preferences to AI — over and over.

That was redundant. So I built this prompt once, saved it, and now I reuse it forever.

---

## What It Does

You give it your solved LeetCode code.
It gives you a complete, beginner-friendly teaching post — instantly.

Every output includes a real-life analogy, plain English concept explanations,
a visual walkthrough, step-by-step code breakdown, complexity analysis,
and a warm note encouraging readers to learn, not copy-paste.

---

## Who It's For

- 🧑‍🏫 Teachers who want to explain solutions, not just share code
- 🐣 Beginners who want to understand the "why" behind every line
- 💼 Developers who want to show communication skills alongside technical skills

---

## How To Use

**Step 1** — Copy the full prompt from [`prompt.md`](./prompt.md)

**Step 2** — Paste it into Claude (claude.ai)

**Step 3** — Follow with your code like this:
```
Problem: [LeetCode problem name or number]

Code:
[your Python3 solution here]
```

**Step 4** — Get your full teaching post ✅
